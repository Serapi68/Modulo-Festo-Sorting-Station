# PROYECTO MODULO FESTO

En este proyecto se busca la realización de un cuadro de control, desde suu programación, montaje y planificación en un modulo FESTO MPS, en este caso Sorting Station.

## Contenido

**Data Sheet**:
- En la carpeta data sheet encontraremos todas las especcificaciones técnicas de los sensores y actuadores que se encuentran en el módulo.

**Información Auxiliar**:
- Hay un pdf de la Universidad Sabana de Colombia que explica de forma breve el funcionamiento de la máquina.
- Se insertan dos videos para la visualización de maniobras.
- Se adjunta manual y un excel con entradas y salidas del módulo.
- Un informe en power point.

**Programación**:

Se adjunta una carpeta comprimida en __.zip__ tener en cuenta:
- TIA PORTAL V20.
- PLC 1214 AC/DC/RLY.
- Se ha hecho la programación en lenguaje KOP (LADDER).
